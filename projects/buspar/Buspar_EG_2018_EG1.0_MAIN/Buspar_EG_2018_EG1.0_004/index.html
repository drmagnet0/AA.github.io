Generalized anxiety disorder (GAD) treatment goals:
When treating GAD, clinicians should try to:3

  • Remove the core psychic symptoms of the disorder3
  • Remove the somatic symptoms3
  • Improve quality of life3
  • Achieve remission or 'wellness'3
  • Reduce disability3
  • Treat comorbid conditions3
